Excellent! Now for the fun part. Let's test the setup with a simple image-to-video workflow using the Stable Video Diffusion (SVD) model we downloaded.

I have created a basic workflow for you in a JSON file. Here's how to use it:

1.  **Download an input image:** Find an image you'd like to animate and save it to the following directory:
    `/mnt/Phocis/User/Delphi/input`
    For example, you could name it `test_image.png`.

2.  **Load the workflow:**
    *   I have written a file named `svd_workflow.json` to your `/mnt/Phocis/User/Demon` directory.
    *   In the ComfyUI interface, click the **'Load'** button (usually in the top right corner) and select the `svd_workflow.json` file from `/mnt/Phocis/User/Demon`.

3.  **Update the workflow:**
    *   Once the workflow is loaded, you will see a set of connected nodes.
    *   Find the **'Load Image'** node and select the image you downloaded in step 1.
    *   Find the **'CheckpointLoaderSimple'** node and make sure `svd_xt.safetensors` is selected.

4.  **Generate the video:**
    *   Click the **'Queue Prompt'** button (usually in the top right corner). ComfyUI will start generating the video.
    *   The output video will be saved in `/mnt/Phocis/User/Delphi/output`.
