To install the HTTP extension for ComfyUI, please follow these steps:

1.  **Open your terminal** or command prompt.
2.  **Navigate to your ComfyUI `custom_nodes` directory.** Based on your earlier clarification, this would be:
    `cd /mnt/Phocis/User/Hot/ComfyUI/custom_nodes`
    *If the `custom_nodes` directory does not exist, you may need to create it first:*
    `mkdir -p /mnt/Phocis/User/Hot/ComfyUI/custom_nodes`
3.  **Clone the repository:**
    `git clone https://github.com/felixszeto/ComfyUI-RequestNodes.git`
4.  **Restart ComfyUI.** This is crucial for the new nodes to be loaded.
